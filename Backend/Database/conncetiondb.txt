# 🧠 Prisma v5 Flow (Understand First)

Whenever you use Prisma, the flow is always:

```
1. Define Models → 2. Migrate DB → 3. Generate Client → 4. Use in Code
```

---

# 🔥 STEP-BY-STEP (Reusable Setup)

---

## ✅ STEP 1 — Install Prisma (in any new project)

```bash
npm install prisma@5 @prisma/client@5
npx prisma init
```

👉 This creates:

* `prisma/schema.prisma`
* `.env`

---

## ✅ STEP 2 — Setup Database Connection

📁 `.env`

```env
DATABASE_URL="postgresql://USER:PASSWORD@localhost:5432/DB_NAME"
```

---

📁 `prisma/schema.prisma`

```prisma
datasource db {
  provider = "postgresql"
  url      = env("DATABASE_URL")
}
```

---

## ✅ STEP 3 — Define Models

Example (for learning):

```prisma
model User {
  id    Int     @id @default(autoincrement())
  name  String
  email String  @unique
}
```

👉 Think of this like your MongoDB schema.

---

## ✅ STEP 4 — Run Migration (Create Tables)

```bash
npx prisma migrate dev --name init
```

✔️ This:

* Creates DB tables
* Creates migration files

---

## ✅ STEP 5 — Generate Prisma Client

```bash
npx prisma generate
```

👉 This creates the query API.

---

## ✅ STEP 6 — Create Prisma Client File

📁 `prismaClient.js`

```js
import pkg from "@prisma/client";
const { PrismaClient } = pkg;

const prisma = new PrismaClient();

export default prisma;
```

---

## ✅ STEP 7 — Use in Backend (CRUD)

### ➤ Create

```js
const user = await prisma.user.create({
  data: {
    name: "Ali",
    email: "ali@gmail.com",
  },
});
```

---

### ➤ Read

```js
const users = await prisma.user.findMany();
```

---

### ➤ Update

```js
await prisma.user.update({
  where: { id: 1 },
  data: { name: "Updated Name" },
});
```

---

### ➤ Delete

```js
await prisma.user.delete({
  where: { id: 1 },
});
```

---

# ⚡ STEP 8 — When You Change Schema

Whenever you modify models:

```bash
npx prisma migrate dev --name update
npx prisma generate
```

# 🚨 Common Mistakes (Avoid These)

❌ Forgetting `.env`
❌ Not running `migrate` after schema change
❌ Not running `generate`
❌ Wrong DB URL
❌ Not disconnecting Prisma
